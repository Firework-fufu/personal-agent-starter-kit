# Skills Index

> Skills are reusable operating procedures for the agent. If a request matches a trigger, read the skill file before acting.

| Skill | Trigger | Output | Path |
|---|---|---|---|
| `skill-lifecycle` | create, update, improve, or review a skill/workflow | new or revised `SKILL.md`, index update, feedback entry | `skills/skill-lifecycle/SKILL.md` |
| `self-review` | important decision, red-team review, plan pressure test | challenge notes, coverage notes, final decision table | `skills/self-review/SKILL.md` |

## When To Add A Skill

Add a skill when:

- the workflow will repeat more than 2-3 times;
- the steps are easy to skip or misorder;
- the output format should stay consistent;
- mistakes should be converted into future checkpoints.

Do not add a skill for one-off tasks.
